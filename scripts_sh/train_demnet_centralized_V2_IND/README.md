Independent centralized counterpart of the FL script V2.

This scripts are similar to the scrips in `train_demnet_centralized_V2` folder, but unlike them, they are "independent".

What I mean is that the scripts in that `train_demnet_centralized_V2` folder were created to do a centralized training keeping the same hyperparameters as previous federated runs (i.e. compare federated and centralized training with the same settings).
This scripts are instead "independent" in the sense that are used to do a centralized training run with the hyperarameters that you prefer.
